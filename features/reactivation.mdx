## What Reactivation Does

Reactivation identifies leads who haven't engaged recently and lets you run targeted re-engagement campaigns. Find leads that went cold and bring them back.

## How It Works

1. Go to **Leads → Re-activation** tab
2. Set your criteria:
   - No reply in X days
   - Last contacted more than X days ago
   - From specific lists or sources (HubSpot, GHL, CSV)
3. Preview matching leads
4. Create a re-activation campaign or export the list

## Re-activation Sources

| Source | Description |
|--------|-------------|
| All leads | Any lead in your workspace |
| HubSpot Import | Leads synced from HubSpot |
| GHL Import | Leads synced from GoHighLevel |
| CSV imports | Leads uploaded via CSV |
| API leads | Leads created via the API |

## Best Practices

- **Wait at least 14 days** before re-activating — give leads time to respond
- **Use a different message** — don't repeat your original outreach
- **Check DNC status** — leads who opted out should not be re-activated
- **Track results** — compare re-activation reply rates vs initial outreach
